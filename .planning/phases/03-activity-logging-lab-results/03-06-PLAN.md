## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| browser -> /api/lab-results(/upload, /trend, /parameters) | Client sends lab payloads, file uploads, and query params (server re-validates — 03-04/03-05) |

| client render | Untrusted lab parameter names / filenames displayed in DOM |

Pin `recharts@3`; not [SUS]/[SLOP], so no blocking legitimacy checkpoint required |
