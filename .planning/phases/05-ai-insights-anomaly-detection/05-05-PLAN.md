| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-05-11 | Elevation of Privilege / Information Disclosure | `/api/ai/*` cache reads | mitigate | `authenticate` on every route + repository functions scope by `req.user!.id`; no cross-user cache read. |
| T-05-12 | Tampering | Prompt injection via tracking notes | mitigate | System prompt instructs the model to treat user fields as data; disclaimer is backend-appended (D-19), not model-trusted. |
| T-05-13 | Information Disclosure | AI narrative cache columns | mitigate | Stored as AES-256-GCM ciphertext via lib/encryption; GROQ_API_KEY/raw bodies never logged. |
