## Screen-by-Screen Contracts

### Screen 1: Pengingat page — 30s polling patch (D-01, D-02, D-03)

**File:** `frontend/app/(app)/pengingat/page.tsx`

No visual changes to the rendered UI. The polling is invisible to the user unless a reminder change is detected from another device.

**Polling logic (add to existing `useEffect` block):**

```typescript
// Polling interval: 30 seconds
// Detect reminder changes via max(updated_at)
// Fire Sonner toast if maxUpdatedAt increases between polls
useEffect(() => {
  if (!accessToken) return;

  let lastMaxUpdatedAt: string | null = null;

  const poll = async () => {
    try {
      const data = await authFetch<Array<{ updated_at: string }>>(
        "/api/reminders",
        accessToken,
      );
      const reminders = Array.isArray(data) ? data : [];
      const maxUpdatedAt =
        reminders.length > 0
          ? reminders.reduce((max, r) =>
              r.updated_at > max ? r.updated_at : max,
              reminders[0].updated_at,
            )
          : null;

      if (lastMaxUpdatedAt !== null && maxUpdatedAt !== null && maxUpdatedAt > lastMaxUpdatedAt) {
        // Reminder changed from another device
        toast("Jadwal pengingat diperbarui dari perangkat lain.");
        setReminderRefreshKey((k) => k + 1); // Trigger list re-render
      }

      lastMaxUpdatedAt = maxUpdatedAt;
    } catch {
      // Silently ignore poll failures — next tick will retry
    }
  };

  // Initial poll on mount
  poll();
  const interval = setInterval(poll, 30_000);
  return () => clearInterval(interval);
}, [accessToken]);
```

**Sonner toast:**
- Neutral type (default — not `.success`, not `.error`)
- Duration: 4000ms
- No action buttons or descriptions
