## Typography

| Role | Size | Font | Weight | Line Height | Phase 2 Usage |
|------|------|------|--------|-------------|--------------|
| Display | 20px | Plus Jakarta Sans | 800 | 1.2 | Delta cairan balance number, hero metric value, fluid ring center value |
| Heading | 14px | Plus Jakarta Sans | 700 | 1.25 | Section titles, card titles, modal titles, form section labels, tab labels, page title in TopBar |
| Body | 12px | DM Sans | 400 | 1.5 | Card body text, list item descriptions, form help text, button labels, error messages |
| Caption | 10px | DM Sans | 500 | 1.4 | Timestamps, status labels, nav tab labels, badge text, timing notes, muted secondary info |

**Weight groups:**

- **Bold group:** Plus Jakarta Sans — 700 for heading/section titles; 800 for display/metric values. Use 700 when in doubt for headings.
- **Regular group:** DM Sans — 400 for body/descriptions; 500 for captions/labels/nav. Use 500 for anything that is a label or needs slight emphasis without being a heading.

**No other font weights are used in Phase 2.** DM Sans 600 is used only for button labels (pill primary buttons — per DESIGN_SYSTEM Button Primary spec). This is the one exception beyond the two-group model; document it as "button label weight."

## Color

### 60/30/10 Model

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `#fdf9f3` (cream) | Page background — `--color-background` |
| Secondary (30%) | `#ffffff` (white) | Cards, bottom nav surface, sidebar surface, modal backgrounds, input backgrounds |
| Accent (10%) | `#2a9d8f` (teal) | See reserved-for list below |
| Warning | `#ef9f27` (amber) | Caution fluid threshold, no-reminder banner, pending/overdue status, HD therapy identity |
| Destructive | `#d4183d` | CAPD effluent Alert Darurat, critical fluid threshold, delete confirmations |

**Accent (`#2a9d8f`) is reserved for exactly these elements:**

1. FAB background gradient endpoint (`linear-gradient(135deg, #7a8c8a, #2a9d8f)`)
2. Primary button background

4. Sidebar active item: left border (`3px solid #2a9d8f`) + label color + subtle bg tint (`#f0faf9`)
5. Delta cairan card: balance number + ring SVG stroke when state is "normal"
6. Form input focus ring (`0 0 0 3px rgba(42, 157, 143, 0.15)`)
7. "Sudah diminum" confirmation checkmark (circle bg in ReminderItem, MedicationLogItem)
8. Pill toggle active state background
9. Desktop "Catat Cairan" sidebar button background

**DeltaCairanCard visual states:**

| State | Condition | Number color | Ring stroke | Background |
|-------|-----------|-------------|-------------|------------|
| No data | 0 entries today | `#7a8c8a` | `rgba(255,255,255,0.2)` | hero gradient |
| Normal | `|delta| ≤ 1000 ml` | `#2a9d8f` | white (on teal gradient) | hero gradient |
| Caution | `|delta| 1001–2000 ml` | `#ef9f27` | white | hero gradient |
| Critical | `|delta| > 2000 ml` | `#d4183d` | white | hero gradient |

> Thresholds are visual guidance only — not medical advice. These are Claude's discretion per RESEARCH.md open question A3. Planner should document in plan task notes.

**DeltaCairanCard layout spec:**

- Outer: hero gradient `linear-gradient(145deg, #f0faf9, #e0f5f2, #cdeee9, #b8e3dc)`, border-radius 16px, padding 16px

- Balance number: 20px Plus Jakarta Sans 800, colored per state, centered in ring
- Sub-label in ring center: "Keseimbangan" (10px DM Sans 500 white/0.8 above number), delta label below (e.g., "-350 ml")
